# **App Name**: Vivah Scroll

## Core Features:

- Rope Pull Animation: Interactive animation where the user pulls a rope to unveil the wedding invitation, creating an engaging and magical opening experience.
- Scroll Reveal: Display wedding invitation details on an unrolling parchment scroll, including couple's names, welcome message, and blessings.
- Interactive Venue Map: Integration with Google Maps to display the wedding venue, allowing guests to easily find and navigate to the location.
- Event Timeline: Dynamic timeline showcasing different wedding events (Haldi, Mehendi, Wedding, Reception) with icons and animations on a mini parchment card.
- RSVP and Blessings: Interactive form for guests to RSVP and send blessings, displayed as floating petals with user messages after submission.
- Music Selection: Ability to replay the background music which underscores the beauty and pageantry.

## Style Guidelines:

- Primary color: Antique gold (#D4A66A) for a royal and luxurious feel.
- Background color: Soft beige (#F5F5DC) for an ivory parchment effect.
- Accent color: Deep maroon (#800000) to add a regal touch.
- Headline font: 'Great Vibes', a calligraphy font, for names and titles.
- Body font: 'Playfair Display', a serif font, for text.
- Use custom icons for each event in the timeline section.
- Subtle animations for unrolling scroll, diya lights, and floating petals.
